# AMRAdapterIronsource

IronSource adapter for AMRSDK. Allows publishers to mediate IronSource interstitial, video and offerwall ads in AMRSDK.

## Requirements

- iOS 12.0+
- Xcode 15+

## Installation

### Swift Package Manager

In Xcode: **File → Add Package Dependencies** and enter the repository URL:

```
https://github.com/admost/AMR-IOS-ADAPTER-IRONSOURCE.git
```

> **Important:** Add `-ObjC` to **Other Linker Flags** in your app target's Build Settings.

### CocoaPods

Add the following to your `Podfile`:

```ruby
pod 'AMRAdapterIronsource'
```

Then run:

```sh
pod install
```
